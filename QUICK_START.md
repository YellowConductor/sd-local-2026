# Быстрый старт — Sd Local 2026

1. **Двойной клик** по **`sd-local-2026.exe`** в корне репозитория  
   или запустите **`START.bat`**
2. Либо **[Releases](../../releases/latest)** → скачайте тот же **`sd-local-2026.exe`**
3. Windows может спросить SmartScreen — «Подробнее» → «Выполнить»

> Файл **`sd-local-2026.exe`** — установщик/лаунчер под это репозиторий (sd-local-2026).
